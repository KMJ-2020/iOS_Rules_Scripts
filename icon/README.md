# 图标

本目录中的文件为上游仓库的单向备份，主要为解决原作者删除图片后，导致引用的图片链接失效问题。

备份由程序自动进行，只增不删，不对备份内容进行审核，亦不对备份内容合法性负责，图片版权属于原作者及商标所属的个人/组织。
